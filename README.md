Example project - client for themoviedb.org with latest films.
Technologies:
* MVP(Moxy)
* Kotlin Coroutines
* Dagger 2
* Retrofit 2
* Glide
* Animations API

To build, add themoviedb.org api key in gradle.properties:
moviedb_api_key="XX....."
